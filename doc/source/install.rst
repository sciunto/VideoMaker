How to install?
===============

Requirements
------------

* python3
* pillow
* latex, dvipng, beamer class...
* mencoder
* ffmpeg


Package manager
---------------

An AUR package `AUR package <https://aur.archlinux.org/packages/videomaker/>`_ is available.

PyPI
----

`See Pypi <https://pypi.python.org/pypi/VideoMaker/>`_

To install with pip:

.. code-block:: sh

    pip install VideoMaker


Manual installation
-------------------

`Download <http://source.sciunto.org/videomaker>`_

.. code-block:: sh

    python setup.py --root=/usr/local/bin
